Desa Wisata Wonocolo

Wonocolo adalah salah satu geosite di Bojonegoro yang dikenal karena warisan migas tradisional. Menurut Dinas Pariwisata Bojonegoro, di sini masih bisa ditemukan sumur minyak tua yang dikelola secara tradisional.

Kayangan Api

Kayangan Api adalah sumber api abadi di kawasan hutan Desa Sendangharjo, Kecamatan Ngasem. Api ini muncul dari gas alam yang menyembur dari dalam tanah dan menyala terus-menerus.

Menurut cerita lokal, tempat ini berkaitan dengan Empu Kriyo Kusumo dari zaman Majapahit.
